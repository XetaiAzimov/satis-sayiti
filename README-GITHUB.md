# satis-sayiti

Small demo project: React + Vite starter with Context-based state and example pages (Home, Products, Contact).

Run locally

```powershell
cd "c:\Users\ACER\Desktop\reakt\qaralama\satis sayiti\reactContext"
npm install
npm run dev
```

Notes

- Context provider: `src/Context/ProviderContext.jsx`
- Routes and pages: `src/AppRouter` and `src/Pages`
- Linting: `npm run lint`
